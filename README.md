### Major tools used
- Java 14
- SQL(PostgreSQL, H2)
- Maven
- JPA(Hibernate)
- Spring (Core, Data, Test)
- Spring Boot
## Secondary tools 
FxWeaver, Lombok
### Deployment
Run jar file as executable(or use IDE)
